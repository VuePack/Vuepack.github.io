# 使用说明

## 警告块
```html
<div class="tip">
    预处理器很强大，但它只是编写 CSS 的辅助工具。出于对扩展和维护等方面的考虑，在大型项目中有必要使用预处理器构建 CSS；但是对于小型项目，原生的 CSS 可能是一种更好的选择。不要肆意使用预处理器！
</div>
```

![警告框](http://7xnjvz.com1.z0.glb.clouddn.com/489a510c-92b9-11e5-9256-341cef6999b6.png)
